![codiko_mobile](https://cloud.githubusercontent.com/assets/4039090/23166214/0792b15c-f840-11e6-82f7-573d0580b82e.jpg)

## Installation
```
git clone git@github.com:Permigo/codiko-mobile.git
cd codiko-mobile

yarn install
react-native link
react-native run-ios
```
